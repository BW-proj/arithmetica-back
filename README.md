# arithmetica-back

Express server for the Arithmetica game backend.
It's a real-time multiplayer game where players answer math problems as quickly as possible in 1vs1 matches.

## Installation

```bash
npm install
```

## Usage

### Build the project

```bash
npm run build
```

### Start the server

```bash
npm run start
```

### Start the server in watch mode

```bash
npm run dev
```

## Development

```bash
npm run dev
```
